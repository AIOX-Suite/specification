## How to Add Your Site

We love showcasing AIOX implementations! Here's how to add yours:

### Option 1: Submit via Pull Request (Recommended)

1. **Fork** this repository
2. **Edit** `community/ADOPTERS.md`
3. **Add your site** using this template:

```markdown
### [Your Site Name](https://yoursite.com)
- **Category:** Media & Entertainment / Technology / E-commerce / etc.
- **Implementation:** WordPress Plugin / Custom / CMS Integration / etc.
- **Since:** Month Year (e.g., November 2025)
- **Description:** 1-2 sentences about your site and how you use AIOX
- **Features Used:** List key AIOX features you're using
```

4. **Submit PR** with title: `docs: add [Your Site] to adopters`

### Option 2: Submit via Issue

1. **Open an issue** titled "Add [Your Site] to Adopters"
2. **Include:**
   - Site name and URL
   - Category/industry
   - Implementation type
   - When you started using AIOX
   - Brief description
   - Features you're using
3. We'll add it for you!

### Option 3: Email Us

Send details to: hello@aiox.io with subject "AIOX Adopter Submission"
